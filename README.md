# DiscordBot with Nodejs

## Information
- Nodejs: >= 16.x
- Discordjs: >= 13.x

## Get Started
- Clone my repo
- Create `.env`
- In `.env`, `TOKEN = <your-bot-token>`
- Edit `config.json`
- Use `npm install` to setup
- Use `npm start` or `node index` to run
- All done

## Usage
- Music: `z!(play [url/id] || list || pause || resume || loop)`
- Simsimi (https://api.simsimi.net/): `z!sim [msg]`
